## Back-End

***install dependencies**: yarn install <br />
**run cmd**: yarn dev <br />
**port**: 3333*

## Front-End Web

***install dependencies**: yarn install <br />
**run cmd**: yarn start <br />
**port**: 3000*

## Mobile

***install dependencies**: yarn install <br />
**run cmd**: yarn start <br />
**IP**: set your Expo or mobile emulator IP*
